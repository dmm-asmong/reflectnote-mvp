복습 폼 — 모바일 레이아웃

복습 입력 폼의 두 섹션("배운 범위" + "내 말로 설명")이 모바일에서 어떻게 보여야 할까요?

A

스텝 방식 (단계별 이동)

1단계(배운 범위)→ 2단계(설명 쓰기)를 탭/버튼으로 분리. 한 화면에 하나의 섹션만 보임. 스크롤 없이 집중 가능.

모바일 — 1단계 화면
1 / 2 — 오늘 배운 범위
과목
과학 수학 영어 국어 사회
다음 →

장점

  • 스크롤 없이 집중
  • 진행 상태 명확

단점

  • 앞으로/뒤로 상태 관리 추가
  • 두 섹션을 동시에 못 봄
B

세로 스크롤 (현재 방식 개선)

두 섹션을 세로로 쌓되, textarea 높이를 줄이고 불필요한 여백을 압축. 스크롤해서 한 번에 전체 폼 파악 가능.
